# 🧠 Taller C# - Ejercicios Prácticos

Este repositorio contiene una colección de ejercicios desarrollados en **C#**, enfocados en reforzar lógica de programación, estructuras de datos y resolución de problemas.

## 📌 Objetivo

El objetivo de este proyecto es practicar y consolidar conceptos fundamentales como:

- Uso de estructuras de control
- Manejo de colecciones (`List`, `HashSet`, `ArrayList`)
- Algoritmos básicos
- Manipulación de cadenas
- Programación modular
- Manejo de errores

## ▶️ Cómo ejecutar el proyecto

1. Abre el proyecto en **Visual Studio** o **VS Code**
2. Asegúrate de tener instalado el SDK de .NET
3. Ejecuta el proyecto:

dotnet run
